# Microservices

This documentation will include all the microservices the API Systen will include

### Order Service
- Create, Update, Delete, View Orders
- Order Statutes
- Publishes Events

### Inventory Service
- Update and View their Current amount of inventory
- View all past inventory
- Reserve Stock (Products)

### Payment Service
- Simulates charges and transactions
- Handles Failures and Secure

### Notification Service
- SMS Messages
- Emails 
- Regular updates on stock available to order, stock unavailable and marketing
